# Excel
This project exploring an extensive tourism dataset containing 5,000 unique visitor profiles to analyze travel trends across five iconic world landmarks: the Eiffel Tower, the Colosseum, Machu Picchu, the Great Wall of China, and the Taj Mahal.
Using Microsoft Excel, this project aims to analyze the dominant age groups among travelers, visitor satisfaction levels, tour duration, and satisfaction scores for each landmark across different rating categories
OBJECTIVES
To find out which age groups are the most common visitors .
To find the aggregate popularity across five major global landmarks by the tourist rating.
To analyze the variance between preferred tour duration and actual tour duration.
To analyze visitor satisfaction scores for each landmark across different rating categories.
